Cumulus API Launchpad Authentication

Cumulus API supports Launchpad as an authentication option. That is, authentication by bringing your own Launchpad token returned from the NASA Launchpad /gettoken endpoint.

Set up the Cumulus API Launchpad Client

Steps for setting up the Cumulus API to use Launchpad authentication can be found at Cumulus API Launchpad Authentication wiki page. Following these steps will allow your client to retrieve a Launchpad sm_token that can be provided to Cumulus API endpoints for authentication.

Cumulus Configuration

  1. Upload PKI certificate to s3

Upload the PKI certificate pfx file to s3, use {{system_bucket}} as bucket name and {{prefix}}/crypto/launchpad.pfx as key, {{system_bucket}} and {{prefix}} are configured in app/config.yml. If a different private key file name other than launchpad.pfx is used, specify it in the launchpad configuration in app/config.yml.

  1. Provide passphrase of PKI certificate in app/.env

LAUNCHPAD_PASSPHRASE=<LAUNCHPAD_PASSPHRASE>

  1. Configure the oauth.provider and oauth.userGroup configuration parameters in app/config.yml. Example configuration of the oauth variables can be found in Cumulus core's example

oauth.provider in app/config.yml must be set to launchpad to use Launchpad authentication.
